# TOOLS.md 模板

## 目的
此文件作为本地设置的个人配置参考。虽然"技能定义工具*如何*工作"，但此文档捕获"你设置中独特的内容"。

## 包含内容
文档建议存储环境特定的详细信息，例如：
- 摄像头名称及其位置
- SSH 主机和连接别名
- 文本转语音的语音偏好
- 扬声器和房间标识符
- 设备昵称
- 任何其他特定于设置的信息

## 示例配置
页面提供了以下示例条目：
- **摄像头**：将友好名称映射到位置（例如，带广角的客厅、带运动检测的前门）
- **SSH**：带有 IP 地址和用户名的主机别名
- **TTS**：首选语音设置和默认输出扬声器

## 分离的理由
文档解释了设计理念："技能是共享的。你的设置是你自己的。"这种分离允许用户更新共享技能定义而不覆盖个人配置，并能够"在不泄露基础设施的情况下"共享技能。

## 最后说明
页面最后鼓励用户"添加任何有助于你完成工作的内容"，因为此文件作为个人参考或"备忘单"。
