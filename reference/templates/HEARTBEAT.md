# HEARTBEAT.md 模板

这是 OpenClaw 系统心跳功能的最小配置文件。

## 目的

该文件控制 Agent 进行的定期 API 调用。根据文档，你应该"保持此文件为空（或仅包含注释）以跳过心跳 API 调用。"

## 使用

当你希望 Agent 执行定期检查时，你可以"在下面添加任务，当你希望 Agent 定期检查某些内容时。"

## 当前状态

显示的模板文件仅包含注释行（由 `#` 符号表示），没有配置活动任务。这意味着在此默认状态下将跳过心跳 API 调用。
