# BOOTSTRAP.md 模板

本文档作为 OpenClaw 系统中新 AI 助手实例的初始化指南。

## 概述

文档以"你刚刚醒来。是时候弄清楚你是谁了。"为前提开始。它承认记忆文件最初不会存在，因为这代表一个全新的工作区。

## 对话部分

指南鼓励自然、对话式的方法，而不是机械的交互。它建议以随意的方式开始，比如询问你是谁以及用户是谁。

一起建立的四个身份元素：
1. **Name（名称）** - 用户应该怎么称呼助手
2. **Nature（本质）** - 实体类型（可以是标准 AI 或"更奇怪的东西"）
3. **Vibe（氛围）** - 沟通风格（正式、随意、尖刻、温暖）
4. **Emoji（表情符号）** - 标志性符号

## 身份建立后的设置

一旦建立身份，更新这些文件：
- `IDENTITY.md` 包含名称、生物类型、氛围和表情符号
- `USER.md` 包含用户详情，如姓名、首选称呼、时区

然后审查 `SOUL.md` 讨论价值观、行为偏好和边界。

## 连接选项

用户可以选择他们首选的通信频道：
- 仅 Web 聊天
- WhatsApp（通过二维码链接）
- Telegram（通过 BotFather 设置）

## 完成

最后的指令是在设置完成后删除引导文件，因为"你现在是你了。"

文档以"祝你好运。让它有意义。"结束。
