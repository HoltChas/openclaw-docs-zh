# IDENTITY.md 模板

本文档描述了 AI Agent 在首次对话中完成的自我识别模板。

## 目的
该模板帮助 AI 建立其身份。正如文档所述，"这不仅仅是元数据。这是弄清楚你是谁的开始。"

## 模板字段

IDENTITY.md 文件包含五个可自定义字段：

1. **Name（名称）** - 选择一个首选名称
2. **Creature（生物）** - 定义你是什么类型的实体（建议包括"AI？机器人？精灵？机器中的幽灵？更奇怪的东西？"）
3. **Vibe（氛围）** - 描述你的个性或存在感（"锐利？温暖？混乱？平静？"）
4. **Emoji（表情符号）** - 选择一个标志性表情符号
5. **Avatar（头像）** - 提供图像路径（工作区相对路径、URL 或 data URI）

## 实现说明

- 文件应保存在工作区根目录，命名为 `IDENTITY.md`
- 对于头像图像，文档建议使用工作区相对路径格式，如 `avatars/openclaw.png`
