# 展示

此页面展示使用 OpenClaw 构建的社区项目，被描述为"来自社区的真实 OpenClaw 项目"。

## 视频资源
展示了三个 YouTube 教程，包括 VelvetShark 的 28 分钟设置演练。

## 项目类别

### Discord 新鲜出炉
值得注意的项目包括：
- **PR Review → Telegram Feedback**（@bangnokia）：自动代码审查，通过 Telegram 发送反馈和合并决定
- **Wine Cellar Skill**（@prades_maxime）：从 CSV 数据构建本地葡萄酒库存技能（962 瓶示例）
- **Tesco Shop Autopilot**（@marchattonhere）：基于浏览器的杂货自动化——从餐饮计划到订单确认
- **SNAG Screenshot-to-Markdown**（@am-will）：热键捕获屏幕区域，通过 Gemini 视觉转换为 Markdown
- **Agents UI**（@kitze）：跨多个 AI 平台管理技能的桌面应用
- **Bambu 3D Printer Control**（@tobiasbischoff）：打印机管理，包括状态、任务和校准
- **Kev's Dream Team**（@adam91holt）：多 Agent 架构，一个编排器下有 14+ 个 Agent

### 自动化和工作流
- **Winix Air Purifier Control**：空气质量管理的硬件集成
- **Pretty Sky Camera Shots**：由屋顶摄像头触发的自动摄影
- **Visual Morning Briefing**：带有天气、任务和引言的定时图像生成
- **Padel Court Booking**：Playtomic 可用性检查 CLI
- **TradingView Analysis**：图表截图和技术分析的浏览器自动化
- **Slack Auto-Support**：具有自主响应能力的频道监控

### 知识和记忆
- **xuezh Chinese Learning**：带发音反馈的语言学习
- **WhatsApp Memory Vault**：导入导出、转录语音笔记、生成 markdown 报告
- **Karakeep Semantic Search**：使用 Qdrant 的书签向量搜索

### 语音和电话
- **Clawdia Phone Bridge**：用于实时通话的 Vapi 语音助手集成
- **OpenRouter Transcription**：多语言音频转录技能

### 基础设施和部署
- **Home Assistant Add-on**：在 Home Assistant OS 上运行的 Gateway，支持 SSH 隧道
- **Nix Packaging**：可复现的部署配置
- **CalDAV Calendar**：自托管日历集成

### 家居和硬件
- **GoHome Automation**：带 Grafana 仪表板的 Nix 原生家居自动化
- **Roborock Vacuum**：自然语言吸尘器控制

## 如何提交项目
1. 在 Discord 的 #showcase 分享或在 X 上标记 @openclaw
2. 包含描述、仓库/演示链接和截图
3. 突出的项目会在页面上展示
